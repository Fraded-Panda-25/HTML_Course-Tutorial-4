Programming Languages

HTML:
The skeleton of a webpage. It structures content using elements like headings, paragraphs, links, images, and tables.
CSS:
The design layer. It controls how HTML looks — colors, fonts, layouts, spacing, animations, and overall style.
Java Script:
The brain of the webpage. It adds interactivity — buttons that respond, forms that validate, sliders, pop-ups, and dynamic content.
Python:
A general-purpose programming language. Used for web development, data analysis, AI, automation, and more. It’s known for being beginner-friendly and powerful.
