---

## 📚 Learning Objectives
By the end of this tutorial, you will:
- Understand **[semantic HTML](ca://s?q=Explain_semantic_HTML)** — tags like `<header>`, `<footer>`, `<section>`, and `<article>`.
- Practice **[text formatting](ca://s?q=HTML_text_formatting_basics)** — bold, italic, underline, superscript, and subscript.
- Create **[lists](ca://s?q=HTML_lists_tutorial)** — ordered, unordered, and nested lists.
- Use **[line breaks and spacing](ca://s?q=HTML_line_breaks_and_spacing)** — `<br>`, `<hr>`, and whitespace handling.
- Combine semantic and formatting tags for a **[structured webpage](ca://s?q=HTML_webpage_structure_basics)**.
